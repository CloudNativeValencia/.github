# Cloud Native Valencia

Cloud Native Valencia is the official CNCF chapter in Valencia, Spain, focused on building a welcoming local community around modern infrastructure and open source technologies.

We host free meetups and community events for people who want to learn, share, and connect through topics such as Kubernetes, DevOps, Platform Engineering, Linux, Observability, Open Source, and the broader Cloud Native ecosystem.[page:4]

## What this organization is for

This GitHub organization is the home of Cloud Native Valencia community projects, meetup planning, shared resources, and anything else that supports the chapter and its events.

You may find event-related repositories, sponsorship material, website content, photos, and other community assets here as the chapter grows.

## Community values

Cloud Native Valencia is built to be safe, inclusive, and accessible. Meetups are free to attend, and everyone is welcome, regardless of background, race, color, or creed.

The goal is simple: create a space where people can learn in public, ask good questions, meet peers, and contribute to the local cloud native community without gatekeeping.
